# Focalize
Focalize is a productivity and project management web platform inspired by Trello. It was designed to help users and teams organize tasks, manage workflows, and improve daily efficiency through an intuitive interface.

Features:

Task Management: Create, update, and delete tasks with ease.

Drag & Drop: Visual organization of tasks across different stages (To Do, Doing, Done).

Responsive Design: Optimized for both desktop and mobile use.

User Authentication: Secure login and account management.

Technologies Used:

Frontend: HTML5 & CSS3 / JavaScript

Backend: Node.js / Express.js

Database: MySQL

Styling: CSS
